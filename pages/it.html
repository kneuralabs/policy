IT & Security
KNL-IT-001 · Last reviewed April 2026

This section explains how to keep our systems, data, and client information safe. You do not need to be a tech expert to follow these rules — they are written in plain language.

Understanding the three types of data we handle

Data sensitivity at a glance

Restricted
Client PII, passwords, API keys · encrypt always · never paste into AI
Confidential
Reports, strategy, salaries · encrypt & don't share outside
Internal
Notes, updates · keep off public sites & social media

Passwords and account security

  • Use a unique, strong password for every work account. A strong password is at least 12 characters and includes letters, numbers, and symbols.
  • Turn on Multi-Factor Authentication (MFA) for every work account that supports it. MFA means you need your password and a second check (like a code on your phone) to log in. This stops hackers even if they steal your password.
  • Never share your password with anyone — not even IT support. Legitimate IT staff will never ask for your password.
  • Use a password manager (ask IT for the approved one) to keep track of passwords safely.

AI tool rules

  • Only use AI tools that appear on the company's approved list.
  • "Shadow AI" means using an unapproved AI tool for work tasks. This is strictly forbidden — even if the tool seems helpful and harmless.
  • Why? Unapproved tools may store your inputs, share data with third parties, or have security flaws we have not assessed.

If something feels wrong, it probably is. Report first, investigate later.

Software and devices

  • Only install software that IT has approved. Unapproved software can contain viruses or spyware.
  • Keep your device updated — install security patches when prompted. These fix known vulnerabilities.
  • Lock your computer whenever you step away from your desk, even for a minute. Use Win+L (Windows) or Ctrl+Cmd+Q (Mac).

Do

  • Use a unique 12+ character password for every work account.
  • Turn on Multi-Factor Authentication (MFA) everywhere it's offered.
  • Lock your screen the moment you walk away.
  • Report lost devices or suspicious emails to IT immediately.
  • Install updates and patches when prompted.

Don't

  • Share your password — not with IT, not with anyone.
  • Click links or open attachments in suspicious emails.
  • Use unapproved AI tools (no "shadow AI").
  • Install random software from the internet.
  • Wait to report a security incident — every minute counts.
For IT vendors and partners: Hold ISO 27001 or SOC 2 certification, provide 24/7 monitoring, run an annual penetration test, and report any security incident within 1 hour.
