IT & Security
KNL-IT-001
  • Types of data: "Restricted" = client personal info, passwords → always encrypt, use MFA, never feed into AI. "Confidential" → encrypt when stored. "Internal" → keep inside the company.
  • Access rules: Turn on multi-factor authentication for every account. Only get the access you really need. Never share your password.
  • AI governance: Use only approved AI tools. Using unapproved "shadow AI" is forbidden.
  • Security incidents: Critical incidents → report within 1 hour. Client data leaked → tell leadership within 2 hours; regulators within 72 hours if required.
Our IT partner must be ISO 27001 or SOC2 certified, monitor systems 24/7, and run a hack test every year.
